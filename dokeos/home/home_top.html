Dokeos LCMS Demo portal

On these pages you can try out the development version of the Dokeos LCMS project. More information can be found at:

  • this Wiki page
  • this forum thread
You can use this demo portal to test the current version of the LCMS project. Please use the forum for any feedback. All data in this demo portal can be cleared at any time without warning.

You can login as John Smith, a biology teacher, using jsmith as username and password.

Once logged in, you'll see a tab "My Repository". This will lead you to your repository manager where you can manage your learning objects. You can also publish some learning objects in John's courses. Not all functionality is working. The announcements tool, calendar tool, description tool, link tool and document tool should work fine.
